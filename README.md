**Технологический стек:**
1. Kotlin
2. Jetpack Compose
3. Coroutines
4. Firebase Firestore + Auth
5. Dagger Hilt
6. RoomORM
7. Compose Navigation

Приложение создаено в соответствии с паттерном проектирования **MVVM**
